📖 Installation Guide

1. Download the bizlink-LP-V1-free.xml file
2. Go to Blogger → Theme → Backup/Restore
3. Click "Choose File" → Upload bizlink-LP-V1-free.xml
4. Click "Upload" → Done!
5. Go to Layout → Edit the "BizLink Home Landing" widget
6. Paste your landing page HTML code

💡 To create a landing page post, simply add the "landing-page" label
