## 📖 Installation Guide

### Step 1: Download the Template
Click the download link above to get the `bizlink-LP-V1-free.xml` file.

### Step 2: Access Your Blogger Dashboard
1. Go to [Blogger.com](https://www.blogger.com)
2. Sign in with your Google account
3. Select the blog where you want to install the template

### Step 3: Upload the Template
1. In the left sidebar, click **Theme**
2. Click the **Backup/Restore** button (☁️ icon with arrow) in the top right corner
3. Click **Choose File**
4. Select the `bizlink-LP-V1-free.xml` file you downloaded
5. Click **Upload**

### Step 4: Verify Installation
- The template will be applied immediately
- Your blog should now display the BizLink template

## 📝 How to Create a Landing Page

### Option 1: Landing Page on Homepage

1. Go to **Blogger Dashboard** → **Layout**
2. Find the widget titled **"BizLink Home Landing"**
3. Click **Edit** (pencil icon)
4. Paste your HTML/CSS landing page code
5. Click **Save**

### Option 2: Landing Page as a Post

1. Go to **Posts** → **New Post**
2. Write your landing page content in HTML view
3. In the right sidebar, under **Labels**, add the label: `landing-page`
4. Click **Publish**
5. The post will automatically display in fullscreen mode (navbar, sidebar, and footer hidden)

## 🎨 Customization Guide

### Changing Colors

Edit the CSS variables in the template:

```css
:root {
  --bl-primary: #667eea;    /* Primary color */
  --bl-secondary: #764ba2;  /* Secondary color */
  --bl-accent: #f56565;     /* Accent color */
  --bl-dark: #2d3748;       /* Dark color */
  --bl-light: #f7fafc;      /* Light color */
}
